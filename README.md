#NephoScale CloudScript
CloudScript is a build language for the NephOS cloud platform.  NephoScale provides blueprints for customers to deploy complex infrastructure with a 1-click installation process. [Learn more.](http://docs.nephoscale.com/#!/cloudscript)

NephoScale is a high performance IaaS cloud hosting service provider.  CloudScript is our tool for DevOps practitioners to orchestrate every resource on NephOS. [Learn more.](http://www.nephoscale.com/nephoscale-platform)
